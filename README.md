# Himalayan Feast & Flavors Restaurant App

A full-stack restaurant ordering system with menu management, user authentication, and admin dashboard.

## Features

- 🍽️ Menu browsing with categories
- 🛒 Shopping cart functionality
- 👤 User authentication (Email + Google OAuth)
- 🏪 Multi-restaurant support
- 👨‍💼 Admin dashboard for menu management
- 📊 Owner dashboard with analytics
- 👥 User management system

## Tech Stack

**Frontend:**
- React 18
- React Router
- Vite
- Bootstrap

**Backend:**
- Node.js
- Express
- MySQL
- JWT Authentication
- Google OAuth 2.0

## Deployment

- Frontend: Vercel
- Backend: Railway
- Database: Railway MySQL

## Local Development

### Backend
```bash
cd project/backend
npm install
node server.js
```

### Frontend
```bash
cd project/frontend
npm install
npm run dev
```

## Environment Variables

See `.env.example` files in backend and frontend folders.
